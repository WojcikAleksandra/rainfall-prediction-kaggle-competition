# Kaggle Playground Series - Season 5 Episode 3

This repository contains code, experiments, and analyses for the [Kaggle Playground Series - Season 5 Episode 3](https://www.kaggle.com/competitions/playground-series-s5e3) competition. The work was completed as a team project as part of university studies. 

## Overview

The objective of this competition is to predict ranifall for each day of the year. In this repository, you'll find:

- Data preprocessing and exploratory data analysis (EDA)
- Feature engineering scripts
- Model training and evaluation code
- Experiment tracking and result logging
